Recent Work

Lightweight Edge RAG System Using Quantized Local LLMs

Built a fully local Retrieval-Augmented Generation (RAG) assistant to answer UK parking regulation queries using quantized small language models running entirely on edge hardware. Engineered a document ingestion and semantic chunking pipeline for Highway Code, CPZ policies, and TMA 2004 regulations with locally generated embeddings. Implemented on-device vector retrieval and citation-grounded prompting to minimize hallucinations and ensure regulation-backed responses. Optimized inference using 4-bit/8-bit quantized models (GGUF) executed via llama.cpp/Ollama for CPU-only deployment. Containerized with Docker and delivered through a lightweight Streamlit interface, ensuring privacy, low latency, and zero cloud dependency.

Behind the scenes, I designed its knowledge layer and agent workflows to continuously learn from campaign data, refine targeting, and adapt outreach strategies. What started as a chatbot evolved into a multi-agent growth engine — one that not only automated engagement but secured new clients by productizing ARIA as a scalable SaaS solution.
